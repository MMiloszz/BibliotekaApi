# BibliotekaApi

Aplikacja REST API została napisana w technologii ASP.NET Core i umożliwia wykonywanie operacji CRUD na encjach Autor oraz Książka.

Projekt był testowany automatycznie przy użyciu dostarczonych testów jednostkowych (HTML + JavaScript). W tym celu utworzono folder wwwroot, do którego dodano pliki testowe pobrane z platformy Moodle. Testy są uruchamiane poprzez otwarcie aplikacji w przeglądarce po jej uruchomieniu.
